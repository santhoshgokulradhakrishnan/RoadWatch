# RoadWatch

RoadWatch is a community-driven pothole reporting system.

## Features
- Report potholes
- Store pothole data in MySQL
- REST API using Spring Boot
- Severity tracking
- Status tracking

## Tech Stack
- Java
- Spring Boot
- MySQL
- Hibernate/JPA
- Maven
